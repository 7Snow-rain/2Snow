# 用户管理系统 V3 修改报告

## 概述

基于原 V3 安全加固版，新增用户注册和搜索功能，同时刻意保留 SQL 注入漏洞以作教学演示。

---

## 新增功能

### 1️⃣ 用户注册 `/register`

- 新页面 `templates/register.html`，含用户名、密码、邮箱、手机号输入框
- 导航栏未登录状态下增加"注册"链接
- POST 提交使用 **f-string 拼接 SQL** 插入到 SQLite 数据库

```python
sql = f"INSERT INTO users (username, password, email, phone) VALUES ('{username}', '{password}', '{email}', '{phone}')"
```

- 注册成功后跳转到登录页并提示"注册成功，请登录"
- 注册的用户密码以 **明文形式** 存储在 `data/users.db` 中

### 2️⃣ 用户搜索（首页）

- 登录后首页显示搜索框
- 通过 GET 参数 `keyword` 接收关键词
- 使用 **f-string 拼接 SQL** 查询

```python
sql = f"SELECT id, username, email, phone FROM users WHERE username LIKE '%{keyword}%' OR email LIKE '%{keyword}%'"
```

- 搜索结果以表格形式显示：ID、用户名、邮箱、手机
- 无搜索结果时显示"无搜索结果"
- 每次搜索在控制台打印执行的 SQL 语句

## 数据库

- **位置**: `data/users.db`
- **表结构**: `users(id, username, password, email, phone)`
- **默认用户**: admin/admin123, alice/alice2025
- **密码存储**: 注册使用明文存储（SQLite），登录仍用 bcrypt（USERS 字典）
